# my-protofolio
This project is a personal interactive portfolio website built using HTML, CSS, and JavaScript. It is designed to showcase personal information, skills, projects, and contact details in a modern and visually engaging way.  
## 🎯 Purpose

This project is created to convert a traditional resume into an interactive digital portfolio that helps in showcasing skills and projects in a more engaging way.

## 📱 Responsive Design

The website is fully responsive and works on:
- Mobile phones
- Tablets
- Laptops
- Desktop screens
